Welcome to the official portfolio of **MUH4RHQ**. This project is a high-performance web application designed to showcase technical expertise in **API Engineering**, **Automation**, and **Applied AI**, all delivered through a futuristic, cyberpunk-inspired user interface.

## 🚀 Key Features

*   **Cyberpunk Aesthetic:** High-contrast dark theme with neon accents, custom CRT effects, and immersive animations.
*   **Hybrid Music Player:** Integrated Spotify and YouTube playback capabilities for an interactive background experience.
*   **Responsive Engine:** Built with a "Mobile-First" philosophy, ensuring seamless performance across all devices (Desktop, Tablet, Mobile).
*   **Interactive UX:** Custom cursor tracking logic and optimized event handling for smooth navigation.
*   **Performance Focused:** Ultra-fast load times powered by the Vite build engine and React 19's concurrent rendering.

## 🛠️ Technical Stack

### Frontend
*   **Framework:** [React 19](https://react.dev/) (Functional Components, Hooks)

*   **Animation:** [Framer Motion](https://www.framer.com/motion/) (Declarative animations & gesture handling)
*   **Build Tool:** [Vite 6](https://vitejs.dev/) (Next-generation frontend tooling)

### Core Expertise (Showcased)
*   **API Engineering:** High-performance API integration and bot development.
*   **Automation:** Specialized in Node.js automation and WhatsApp API ([Baileys](https://github.com/WhiskeySockets/Baileys)).
*   **Architecture:** Clean, modular code structure for scalability and maintainability.

## ⚙️ Getting Started

### Prerequisites
*   Node.js (v18 or higher)
*   npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/Har404-err/Portofolio.git
    cd Portofolio
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run development server**
    ```bash
    npm run dev
    ```
    Open your browser and navigate to `http://localhost:5173`.

4.  **Build for production**
    ```bash
    npm run build
    ```

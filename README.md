## 🚀 Key Features

- **Cyberpunk Aesthetic** — High-contrast dark theme with neon cyan/violet accents, CRT scanline effects, and immersive animations.
- **Animated Loading Screen** — Self-drawing "M" brand mark rendered with SVG stroke animation and neon glow (pure Framer Motion, zero extra dependencies).
- **Hybrid Music Player** — Integrated Spotify and YouTube playback for an interactive background experience.
- **Custom Cursor Engine** — Dot + follower cursor with hover-aware interactivity, working on both desktop (mouse) and mobile (touch).
- **Responsive by Design** — Mobile-first philosophy ensuring seamless performance across Desktop, Tablet, and Mobile.
- **Performance Focused** — Ultra-fast load times powered by Vite 6 and React 19 concurrent rendering.

## ⚙️ Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18 or higher
- npm (or yarn / pnpm)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kyiov/portofolio-.git
   cd portofolio-
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   ```
   Open your browser and navigate to **`http://localhost:3000`**.

4. **Build for production**
   ```bash
   npm run build
   ```
